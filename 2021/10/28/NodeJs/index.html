fs模块

fs.stat()

fs.stat(path[options],callback)

path:string|buffer|URL

options:Object

callback:Function

  • err
  • stats

fs.mkdir()

  • path:string|buffer|url
  • options:Object|integer
    • recursive:boolena默认false
    • mode:string|integer
  • callback:Function
    • err

异步的创建目录，回调给出一个可能的异常和创建第一个目录路径。当recursive为true时，如果没有创建目录则path仍为undefined

fs.writeFile()

  • file:string|buffer|url|integer
  • data:string|buffer|typeArray|dataviwe|object
  • options:Object|string
    • encoding:string|null
    • mode:integer
    • flag:string
    • signal
  • callback:function
    • err

当file是文件名时，将数据异步地写入文件，如果文件已存在则替换该文件。data可以是字符串或者缓冲区。

fs.appendFile()

  • path:string|buffer|url|number
  • data:string|object
  • options:object|string
    • encoding:string|null
    • mode:integer
    • flag：string
  • callback：Function
    • err

异步地将数据追加到文件，如果该文件不存在，则创建该文件。data可以是字符串或者buffer

fs.readFile()

  • path:string|buffer|url|integer
  • options:object|string
    • encoding:string|null
    • flag:string
    • signal:abortsignal
  • callback:Function
    • err
    • data

异步地读取文件的全部内容

async/await

简化异步编程，让异步看起来像同步

async function foo(){ await 1}

等价于

function foo(){return Promise.resolve(1).then(()=>undefined)}

async/await的优势在于处理then链。

async的优点

  • 内置执行器
  • 更好的语义
  • 更广的适用性
